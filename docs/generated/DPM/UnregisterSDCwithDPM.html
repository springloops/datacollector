Unregister Data Collector from DPM

You can unregister a Data Collector from DPM when you no longer want to use that Data Collector installation with DPM. When you restart an unregistered Data Collector, you use your Data Collector user account to log in.

You can unregister a Data Collector from the Data Collector console or from DPM. For a Cloudera Manager installation, you must use both DPM and Cloudera Manager to unregister a Data Collector. For instructions on unregistering from DPM and Cloudera Manager, see the DPM online help.

Unregistering from Data Collector

You can unregister a Data Collector from DPM using the Data Collector console.

  1. In DPM, stop all jobs running on the Data Collector.
  2. In Data Collector, click Administration > Disable DPM.
    The Disable DPM Confirmation dialog box appears.
  3. To disable Data Collector from working with DPM, click Yes.
  4. Restart the Data Collector in one of the following ways:
    • If you started the Data Collector manually from the command line, click Restart Data Collector in the Disable DPM window.
    • If you started the Data Collector as a service, you must use the command line for restart. Click Cancel in the Disable DPM window, and then run the following command: 
      service sdc restart
